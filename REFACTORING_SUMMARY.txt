================================================================================
MÉTODO PRINCIPAL: EscalaService.generate_schedule()
================================================================================

Assinatura:
  def generate_schedule(
    month: int,
    year: int,
    respect_couples: bool = True,
    balance_distribution: bool = True,
  ) -> Tuple[bool, str, List[Dict[str, str]]]

Retorno:
  • success (bool): Geração bem-sucedida
  • message (str): Erros, avisos ou sucesso
  • schedule (list[dict]): Escala com {evento, data, dia_semana, horario, squad, membro}

Fluxo Interno:
  1. Validar entrada (helpers)
  2. Coletar eventos do BD (fixo, sazonal, eventual)
  3. Para cada evento:
     a. Buscar membros disponíveis (verifica restrições)
     b. Processar casais (se respect_couples=True)
     c. Selecionar por patente + equilíbrio + regra recruta
  4. Retornar escala + lista de conflitos

Exemplos de Uso:
  service = EscalaService()
  success, msg, escala = service.generate_schedule(3, 2025, True, True)
  if success:
      print(f"✅ {msg}")
      for entry in escala:
          print(f"{entry['data']} - {entry['evento']} - {entry['membro']}")
  else:
      print(f"❌ {msg}")
